# Personal Portfolio Website

This is a clean and modern personal portfolio website built with HTML, CSS, and JavaScript. It showcases my skills, education, experience, and projects as a Web Developer and App Developer.

## Features

- **Responsive Design:** Fully responsive layout that works on desktops, tablets, and mobile devices.
- **Smooth Navigation:** Fixed navigation bar with active link highlighting as you scroll.
- **Hero Section:** Professional introduction with profile image and contact information.
- **About Section:** Personal bio, skill tags, and downloadable CV button.
- **"What I Do" Cards:** Visual representation of services (Web Development, Power BI, Flutter).
- **Skills Section:** 
  - Technical skills with animated progress bars
  - Professional skills with circular progress indicators
- **Education & Experience:** Timeline-style cards showing academic background and work experience.
- **Portfolio Gallery:** Filterable project categories (Web Design, Mockups, Flutter, Python, Dashboards).
- **Contact Section:** Contact form with name, email, and message fields.
- **Social Media Links:** Connect via Twitter, GitHub, and Dribbble.

## Built With

- **HTML5** - Structure and content
- **CSS3** - Styling, animations, and responsive design
- **JavaScript** - Interactive features, scroll animations, portfolio filtering
- **Font Awesome** - Icons throughout the site
- **Google Fonts** - Typography
